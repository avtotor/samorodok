# Самородок

Веб-приложение для автоматической оценки PDF-резюме под выбранную вакансию: текст резюме сравнивается с описанием вакансии через локальную модель **Ollama** (без отправки данных во внешние API). Результат — балл соответствия, признак «подходит / не подходит» и краткое обоснование; история оценок сохраняется локально.

---

## Установка

### Требования

- **Node.js** 20 или новее  
- Установленный **Ollama** и скачанная модель (имя задаётся в `.env`, например `ollama pull gemma2`)

### Шаги

1. Склонируйте репозиторий и перейдите в каталог проекта.

2. Создайте файл окружения:

   ```bash
   copy .env.example .env
   ```

   В **Linux/macOS**:

   ```bash
   cp .env.example .env
   ```

   Отредактируйте `.env`: как минимум `OLLAMA_MODEL`, при необходимости `OLLAMA_BASE_URL`, `OLLAMA_NUM_CTX`, пути к папкам.

3. Установите зависимости:

   ```bash
   npm install
   ```

   или

   ```bash
   pnpm install
   ```

4. Подготовьте каталоги (если их ещё нет):

   - **`resumes/`** — PDF-резюме  
   - **`vacancies/`** — описания вакансий в файлах `.vc` или `.txt`

5. Запустите Ollama (чтобы слушал адрес из `.env`, обычно `http://127.0.0.1:11434`).

6. Запустите сервер:

   ```bash
   npm start
   ```

7. Откройте в браузере адрес из консоли (порт по умолчанию задаётся в `.env`, чаще всего `http://localhost:3000`).

Подробности переменных окружения — в комментариях внутри **`.env.example`**.
